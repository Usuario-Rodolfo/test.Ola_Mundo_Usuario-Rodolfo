# Ola Mundo Usuario Rodolfo

![Linguagem](https://img.shields.io/badge/Linguagem-C-blue)
![Compilador](https://img.shields.io/badge/Compilador-GCC-yellow)

Programa em **C** que demonstra a estrutura básica de um programa, declarando uma variável e exibindo uma saudação personalizada no terminal:

Olá, Usuario-Rodolfo!


## Estrutura do Projeto

| Pasta/Arquivo                        | Descrição                     |
|-------------------------------------|-------------------------------|
| src/                                 | Pasta do código-fonte         |
| src/ola_mundo_usuario_rodolfo.c      | Código fonte em C             |
| README.md                            | Este arquivo                  |
| LICENSE                              | Arquivo de licença (opcional) |

## Como Compilar e Executar

| Passo | Comando / Ação                                                   |
|-------|-----------------------------------------------------------------|
| 1     | Abra o terminal na pasta do projeto                              |
| 2     | Compile o programa: <br>```bash<br>gcc src/ola_mundo_usuario_rodolfo.c -o ola_mundo_usuario_rodolfo<br>``` |
| 3     | Execute o programa: <br>```bash<br>./ola_mundo_usuario_rodolfo      # Linux/macOS<br>ola_mundo_usuario_rodolfo.exe    # Windows<br>``` |

## Saída Esperada

Olá, Usuario-Rodolfo!


## Requisitos

- Compilador C (`gcc`) instalado e configurado no PATH
